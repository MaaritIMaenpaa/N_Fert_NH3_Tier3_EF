# N_Fert_NH3_Tier3_EF

## Sub-folders

### data 
*Raw*-folder contains original database in excel and csv-format. *Wrangled*-folder contains RDS-objects with data in the format compatible with the analyses.

### scripts
Script for data wrangling, used to access the original data, and use that to create the cleaned up versions of the database in RDS-format. Script for comparing different modeling techniques and tracking the reasons for differences in estimates between different forms of the models. 

### reports
R markdown documents containing the analysis for 1. All data, 2. Outdoor only data, and 3. Outdoor only, micromet data. Each document details the modeling process for each data subset. 

### output
Currently nothing. Figures/final reports/tables can be added here.
