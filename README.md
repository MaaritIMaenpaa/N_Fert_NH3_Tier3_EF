# N_Fert_NH3_Tier3_EF

All code presented as an R-markdown file as well as a corresponding HTML document. The models presented for all data, outdoor only data and only using data measured with Micromet method.

## Sub-folders

### Sensitivity_explorations
Comparing model results using alternative modeling approaches.

### Alternative models
Explorations of the models using a cuberoot transformation.
